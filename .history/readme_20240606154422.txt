The plugin collects all the plugins from the same *author* together and hides them from the plugins page. Manage them on the plugins page toggle container or on the settings page of the plugin.

== Description ==

If you have a lot of plugins from the same author, this plugin helps you to keep them better organized. 
After switching the plugin on, you can set the author's name on the plugin's settings page located at Grouped Plugins menu item. 
All plugins from this author will be hidden from the plugins view. However, you can manage and see all the plugins on the menu page or on the plugins page
under the plugin name in the toggle container. Switch them on or off with the toggle switches. You will also be able update your plugins in the toggle container.
You can find the plugin author -field used by this plugin on the "Plugins" -menu page in the "Description" -column after the version number.

== Installation ==

1. Use the "Add New Plugin" -button on the plugins page or upload the folder to your WordPress /wp-content/plugins/ -directory.
2. Activate the plugin on the plugins -page of your WordPress admin.

The plugin works automatically without configuration, but you need to set the name of the author for the grouped plugins on the plugin settings page.
